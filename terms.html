01 Definições

02 Descrição do serviço

O EuPosto é uma aplicação web que permite ao Utilizador publicar vídeos em múltiplas redes sociais — YouTube, Instagram, TikTok e Facebook — de forma simultânea, através de um único interface.

A aplicação funciona exclusivamente no browser do Utilizador, sem servidor de backend próprio. Os dados (tokens de API, histórico, configurações) são armazenados localmente no browser do Utilizador. O EuPosto atua como intermediário técnico entre o Utilizador e as APIs das Plataformas de Terceiros.

O Serviço é disponibilizado a título gratuito durante a fase de testes. O Responsável reserva-se o direito de introduzir funcionalidades pagas no futuro, com aviso prévio adequado.

03 Acesso e conta

O acesso ao EuPosto requer a criação de uma conta local (armazenada exclusivamente no browser) ou a autenticação via Google. A conta é pessoal e intransmissível.

O Utilizador é responsável por:

O Utilizador deve ter pelo menos 13 anos para utilizar o Serviço. Para utilizadores entre 13 e 18 anos, é necessário o consentimento de um encarregado de educação.
